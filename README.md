# 🏦 Royal Microfinance Website

React + Tailwind CSS se banaya gaya professional microfinance website.

## 🚀 Setup & Run Karne Ka Tarika

### Step 1: Dependencies Install Karein

```bash
npm install
```

### Step 2: Development Server Start Karein

```bash
npm run dev
```

Browser mein open karein: `http://localhost:5173`

### Step 3: Production Build

```bash
npm run build
```

```
royal-microfinance/

└── src/
    ├── main.jsx
    ├── App.jsx
    ├── index.css
    └── components/
        ├── Navbar.jsx       ← Sticky Navbar with top bar
        ├── Hero.jsx         ← Hero section with loan card
        ├── About.jsx        ← About company section
        ├── Services.jsx     ← 6 loan service cards
        ├── Stats.jsx        ← Stats counter bar
        ├── WhyUs.jsx        ← Why Choose Us section
        ├── Team.jsx         ← Team members section
        ├── Testimonials.jsx ← Client reviews
        ├── Contact.jsx      ← Contact form + info
        └── Footer.jsx       ← Footer with all links
```

## ✨ Sections Included

1. **Navbar** - Sticky navbar with top info bar + mobile menu
2. **Hero** - Full-screen hero with loan card visual
3. **About** - Company description with stats
4. **Services** - Personal, Group, Business, Housing, Education, Agriculture loans
5. **Stats** - Client count, loan amount, etc.
6. **Why Us** - 6 key advantages
7. **Team** - Team member cards
8. **Testimonials** - Client reviews (NEW ✨)
9. **Contact** - Loan application form + contact info
10. **Footer** - Links, social media, contact
